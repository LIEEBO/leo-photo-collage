# Leo Photo Collage 🔲

一个纯前端的智能拼贴画工具，支持多种布局模板、自由拖拽、缩放、旋转照片。

## 🚀 在线访问

部署在 GitHub Pages：`https://<你的用户名>.github.io/<仓库名>/`

## 🛠️ 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 📦 技术栈

- React 19
- TypeScript
- Vite 7

## 📄 部署

推送到 `main` 分支会自动触发 GitHub Actions 部署到 GitHub Pages。
手动部署也可以通过 GitHub Actions 的 `workflow_dispatch` 触发。
